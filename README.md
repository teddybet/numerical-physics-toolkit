# PhysicsPy

PhysicsPy is a Python toolkit for numerical methods and interactive physics simulations.

The project is designed to connect applied mathematics, scientific computing, and physics. It implements numerical algorithms from scratch and applies them to problems such as projectile motion, harmonic oscillators, planetary orbits, relativity, cosmology, and quantum mechanics.

## Project Goals

- Implement core numerical methods from scratch
- Build physics simulations using Python
- Create clear visualizations
- Write clean and tested code
- Develop a portfolio project for scientific computing and applied mathematics

## Planned Modules

- Numerical methods
- Classical mechanics
- Relativity
- Cosmology
- Quantum mechanics
